Descubra seu IMC

O Índice de Massa Corporal (IMC) é uma fórmula utilizada para calcular se o peso está de acordo com a altura.

Fórmula: IMC = peso (kg) / altura² (m²)

Exemplo:

IMC = 70 / (1.75 * 1.75) = 22.86
